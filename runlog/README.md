# 달려 🏃

**혼자 뛰는 사람을 위한 달리기 기록 앱.**
내가 뛴 걸 저장하고, 페이스가 나아지는지 그래프로 보고, 오늘 몸 상태·건강에 맞는 달리기 강도를 추천받아요.

▶ 바로 열기 / 설치: **https://diterun.github.io/playwithme/runlog/**

---

## 📲 폰에 설치하기

브라우저로 위 주소에 접속한 뒤:

- **안드로이드(Chrome)**: 상단에 뜨는 **설치** 배너를 누르거나, 메뉴(⋮) → **앱 설치 / 홈 화면에 추가**
- **아이폰(Safari)**: 아래 **공유 ⬆️** → **홈 화면에 추가**
- 설정(⚙️) 탭의 **앱 설치** 카드에서도 안내를 볼 수 있어요.

설치하면 홈 화면의 **달려** 아이콘으로 앱처럼 열리고, **인터넷이 없어도** 기록·조회가 됩니다.

> 💡 설치 후에는 서버·PC 없이 폰 안에서만 돌아가요. 기록도 폰 안에 저장돼요.

## 화면 구성 (아래 탭 4개)

### 🏃 기록

**오늘·이번 주 현황과 달린 기록 목록**이 있는 홈이에요.

- **＋ 버튼**(오른쪽 아래)으로 새 기록 추가:
  - **날짜**
  - **거리(km)** — 소수점 셋째 자리까지 (예: `2.797`)
  - **시간** — 분 + 초 따로 입력 (예: 25분 30초)
  - **어디서 뛰었어요?** — 🌳 실외 / 🏃 트레드밀 / ⚙️ 무동력 중 선택 (마지막에 고른 게 다음에 기본으로 떠요)
  - **느낌**(😀🙂😮‍💨😣)과 **메모**(선택)
  - 저장하면 **페이스(1km당 시간)가 자동 계산**돼요.
- 기록마다 **✏️ 수정 / 🗑️ 삭제** 가능.
- 새 기록이 이전 최고 기록을 깨면 저장할 때 **"🎉 신기록!"** 축하가 떠요.
- **이번 주** 카드:
  - **🔥 연속 며칠 달렸는지(스트릭)**
  - 주간 목표를 정했다면 **횟수·거리 진행률 막대**
- 맨 위 **🎯 목표** 카드: 목표 페이스에 얼마나 다가갔는지 보여줘요.

### 📊 통계

기록이 쌓일수록 볼거리가 많아져요.

- **개인 기록 🏅** — 최장 거리 · 최장 시간 · 최고 페이스
- **평균 페이스 추이** — 회차별 페이스를 선으로. **위로 갈수록 빠른 페이스**이고, 왼쪽 **분 단위 눈금선**으로 몇 분대인지 한눈에 봐요.
  - **＋ / －** 로 한 번에 보는 기록 수를 조절(최근 5개 ↔ 전체).
  - 위쪽 **필터 칩**(전체/실외/트레드밀/무동력)으로 **같은 환경끼리만 비교** — 환경이 다르면 페이스가 달라지니까요.
- **월별 거리** — 최근 6개월 막대그래프
- **요일별 패턴** — 무슨 요일에 많이 뛰는지
- **월별 달력** — 뛴 날에 👟 도장. `◀ ▶` 또는 **제목(년.월)을 눌러** 원하는 년·월로 이동. 그 달의 거리·횟수·평균 페이스 요약도 함께. (앱을 다시 열면 오늘 달로 돌아와요.)

### ❤️ 건강

**오늘 얼마나 뛰는 게 좋을지 추천**받는 곳이에요.

1. 오른쪽 아래 **📋 건강 프로필** 버튼으로 내 상태를 한 번 등록 (심장·고혈압·허리디스크·천식·당뇨·관절·과체중·임신 등, 항목마다 세부 정도까지).
2. **오늘 몸 상태** 슬라이더 4개(수면·몸·통증·의욕)를 맞추고 **오늘 추천 받기**.
3. 결과:
   - 위험 항목(심장 등)이 있으면 강도를 정해주는 대신 **"의사 상담 먼저"**를 안내해요.
   - 아니면 컨디션·건강·최근 운동량을 종합해 **오늘 강도(쉬기~높게)**를 추천하고, 내 상태에 맞는 **주의사항**을 함께 보여줘요.
   - 최근 **여러 날 연속** 달렸으면 자동으로 **"오늘은 쉬어요"** 쪽으로 낮춰줘요(과훈련 방지).
   - 문구는 매번 조금씩 다르게 나와요.

### ⚙️ 설정

- **앱 설치** 안내
- **목표** — 목표 거리·시간 (→ 목표 페이스). 기록 화면에 진행 표시.
- **주간 목표** — 주당 횟수·거리. '이번 주' 카드에 진행률.
- **화면 테마** — ☀️ 밝게 / 🌙 어둡게
- **데이터**
  - **백업(JSON)** — 전체를 파일로 내보내기
  - **불러오기** — 백업 파일로 복원 (기기 이전 시)
  - **CSV 내보내기** — 엑셀에서 열어보기용
  - **전체 기록 삭제**

## 💾 내 기록은 안전한가요?

- 기록은 **이 폰(브라우저)에만** 저장돼요. 서버로 보내지 않아요.
- **앱이 업데이트돼도 이전 기록은 그대로 남아요.**
- 다만 **폰을 바꾸거나 브라우저 데이터를 지우면** 사라지니, 가끔 **설정 → 백업(JSON)** 으로 파일을 저장해두는 걸 권해요. 새 기기에서 **불러오기**로 그대로 옮길 수 있어요.
