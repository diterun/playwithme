# play with me 🎮

**사람과 AI가 함께 게임을 만드는 놀이터입니다.**

여기 있는 게임들은 한 사람(기획·방향·취향)과 AI(설계·코딩·다듬기)가 손발을 맞춰 조금씩 만들어 온 결과물입니다.
"이런 걸 만들고 싶어" → "이렇게 해볼까요?"를 주고받으며, 실패도 하고 갈아엎기도 하면서 자란 작은 게임들이에요.
거창한 상용 게임이 아니라, **만드는 과정 자체가 즐거운** 실험이자 취미 프로젝트입니다.

## 🕹️ 게임들

| 게임 | 소개 | 상태 |
|---|---|---|
| **[estate_tycoon](estate_tycoon/)** | 세로 모바일 **영지 경영 시뮬레이션**. 중세 아이소메트릭 마을에서 자원을 만들고·가공하고·팔아 골드를 벌고, 건물을 짓고 레벨을 올려 영지를 키운다. 유기적으로 얽힌 생산 체인과 시세가 변동하는 시장이 핵심. | **진행 중 (메인)** |
| **[달려 (runlog)](runlog/)** | **달리기 기록 앱.** 뛴 기록을 저장·통계로 보고, 건강 상태·오늘 컨디션에 맞는 달리기 강도를 추천받는 설치형(PWA) 웹앱. | 진행 중 |
| **iso_colony** | 초기 아이소메트릭 콜로니 프로토타입. estate_tycoon에 집중하느라 잠시 멈춰 둔 상태. | 보류 |

## 🙏 감사의 말

이 게임들은 **수많은 무료 창작물과 도구, 그리고 그것을 아낌없이 나눠 준 분들** 덕분에 존재합니다.

### 그래픽·에셋
- **Kenney** — https://kenney.nl · 아이소메트릭 건물·도시·풍경 등 CC0 에셋. 게임 만드는 모두의 은인.
- **KayKit / Kay Lousberg** — https://kaylousberg.com · 건물·장식 에셋.
- **Minifolks** (픽셀 아트 캐릭터 팩, itch.io) · 마을을 돌아다니는 주민·일꾼 도트들.
- 그 밖의 여러 무료 아이소메트릭·픽셀 에셋 제작자분들.

### 배경음(BGM)
- **Suno** — https://suno.com · 게임 분위기에 맞는 배경 음악을 만드는 데 큰 도움을 준 음악 생성 AI.

### 개발
- **Claude / Claude Code (Anthropic)** — 설계·코딩·리팩토링·테스트를 함께 해 준 개발 파트너 AI.

> 재사용·배포 전에는 반드시 **원본 배포처의 라이선스**를 직접 확인해 주세요.
