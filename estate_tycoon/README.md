# 🏰 estate tycoon — 영지 경영 시뮬레이션

중세 아이소메트릭 마을에서 **자원을 만들고 → 가공하고 → 팔아서** 골드를 벌고,
건물을 짓고 레벨을 올려 영지를 키우는 세로 모바일 게임입니다. 전투는 없습니다. **짓고 키우는 재미**가 전부예요.

## ▶ 플레이 방법

브라우저에서 아래 주소로 접속하세요 (PC·모바일 모두 됩니다):

### 👉 https://diterun.github.io/playwithme/estate_tycoon/

> 📱 **모바일은 세로 화면**에 맞춰져 있습니다. 홈 화면에 추가하면 앱처럼 쓸 수 있어요.

## 💾 기기에 저장해서 플레이 (추천)

인터넷 없이도, 접속이 끊겨도 이어서 하려면 **게임을 파일 하나로 저장**하세요.

1. 게임 접속 후 오른쪽 위 **⚙️ 옵션** 열기
2. **"로컬 게임으로 저장 → 💽 HTML 파일 만들기"** 누르기 (게임 그림·진행상황이 통째로 담긴 HTML 파일이 내려받아집니다)
3. 그 **HTML 파일을 열면** 인터넷 없이 바로 플레이 가능 — 오프라인에서도 100% 동작합니다.

> HTML 파일 만들기는 위 GitHub 주소로 접속 중일 때만 됩니다.
> 진행상황은 **자동저장**되지만, 브라우저 데이터를 지우면 사라질 수 있으니 가끔 **옵션 → 파일로 내보내기(JSON)**로 백업해 두면 안전합니다.

## 🎯 목표

**영주성**을 키우는 게 큰 축입니다. 영주성 레벨이 모든 건물의 레벨 상한이자, 새 건물·기능의 해금 열쇠예요.
자원을 생산·판매해 골드를 모으고, 건물을 올리고, 땅을 넓혀 영지를 점점 화려하게 만들어 갑니다.

## 🕹️ 조작

- **지도 이동**: 한 손가락 드래그 (PC는 마우스 드래그)
- **확대/축소**: 두 손가락 핀치 (PC는 마우스 휠)
- **건물 탭**: 정보 창 열기 (생산·판매·레벨업)
- **건물 길게 누르기** 또는 왼쪽 위 **🔧 편집**: 이동·회전·지형 꾸미기
- 아래 탭바: **🏗️ 건설 · 🏠 영지 · 📦 자원** / 오른쪽 위 **⚙️ 옵션**(저장·배경음)

## 📦 핵심 플레이

- **생산**: 생산건물(제련소·광산·대장간·농장·방앗간·교회 등)을 탭 → 레시피의 **소량 / 중량 / 대량** 중 선택해 대기열에 넣습니다.
  - 소량 = 빠르고 적게 / 중량 = 적당 / 대량 = 오래 걸리지만 많이.
  - **부지런히 소량을 반복하는 게 효율이 제일 좋고**, 대량은 오래 자리 비울 때(방치·오프라인) 유리합니다.
- **생산 체인**: 원료 → 가공품 → 완성품이 유기적으로 얽혀 있습니다. (예: 밀·감자 → 가축 → 소시지·우유 → 치즈)
- **시장**: 생산할 수 있는 품목을 **사고팔 수 있습니다**.
  - **시세가 시간마다 오르내립니다**(인플레·디플레). 다른 품목과 연동되기도 하고, 이유 없이 튀기도 해요.
  - 각 품목의 **최근 시세 그래프(▲/▼)**를 보고 쌀 때 사고 비쌀 때 팔아 보세요.
  - 시장 레벨이 오르면 **거래 스프레드가 좋아집니다**(살 때 더 싸게, 팔 때 더 비싸게).
- **집**: 주민이 세금(골드)을 냅니다. 건물 위 **💰** 표시를 누르면 모든 집에서 한 번에 수거돼요.
  높은 레벨의 집은 빵·치즈 같은 **가공식품을 공급하면 세금이 더 늘어납니다.**
- **영지 개간**: 맵의 빽빽한 숲(4×4 구역)을 탭하면 골드를 내고 개간해 건설 가능한 땅을 넓힙니다.
- **오프라인 진행**: 게임을 꺼 둬도 생산·판매·세금·시세가 계속 흘러갑니다. 다시 켜면 그동안 쌓인 결과가 반영돼요.

## 📖 이야기 · 튜토리얼

처음 시작하면 **영주와 동료들이 등장하는 대화**로 게임을 익히게 됩니다. 이야기는 진행에 맞춰 자연스럽게 이어져요.

- **시작**: 영주·검사·왕자의 대화 → 왕자가 **시장** 사용법 안내
- **생산**: 나무 제련소나 광산을 처음 열면 스피어맨이 **생산** 설명, 소드맨과 담당 건물 소개
- **건물 해금**: 영주성 레벨이 올라 새 건물이 열릴 때마다(작은 집·농장·대장간·방앗간·교회) 담당 동료가 나와 소개
- **진짜 시작**: 영주성 **11레벨**에 모든 건물이 열리면, 영주와 모든 동료가 모여 앞날을 다짐하는 이야기로 마무리

> 모든 대화에는 **[건너뛰기]** 버튼이 있어 언제든 넘길 수 있습니다. 게임을 **초기화**하면 이야기도 처음부터 다시 볼 수 있어요.
